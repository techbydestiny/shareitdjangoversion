# 📤 ShareIt (Django Version)

A web application inspired by **SHAREit**, built with **Django** for the backend and a responsive **HTML/CSS/JavaScript** frontend.  
This version enables users to upload, share, and download files seamlessly through a web interface.

---

## 📖 About
Unlike the front-end-only version, this project includes a **Django-powered backend** that handles:  
- File uploads and storage  
- Secure file sharing  
- Download management  

The app provides a simple, browser-based alternative for sharing files without external services.

---

## ✨ Features
- ✅ Built with **Django** (Python backend)  
- ✅ Upload and share files via unique download links  
- ✅ Simple and responsive UI  
- ✅ Secure file handling  
- ✅ Works on desktop and mobile browsers
